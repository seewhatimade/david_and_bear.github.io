# David and the Lost Bear!

Help David find his baby bear and bring it home before the hungry dinosaur gets to it!

https://github.com/seewhatimade/david_and_bear.github.io/

## How to Play

1. **Search the bushes** scattered around the neighborhood to find the hidden bear.
2. Once you find it, David **picks it up** automatically — now run for home!
3. **Get back to the front door** before the dinosaur catches you.
4. Deliver the bear and a new round starts with a new one to rescue.

## Controls

**Keyboard:**
- Arrow keys (or WASD) to move
- P or Esc to pause

**Mobile:** On-screen D-pad in the bottom-left corner.

## Watch Out

- **The dinosaur** sniffs around the bushes looking for the bear. If it finds the bear first, it's game over.
- **Once you pick up the bear**, the dinosaur chases David directly — don't let it catch you!
- **Mom and Dad** come looking for David if a round takes too long (40 seconds). If they reach him, he gets taken home and the game ends.
- The dinosaur gets a little faster each round.

## Worlds

The game shifts scenery as you rescue more:

1. **Land** — rescue 5 bears from bushes while dodging a dinosaur.
2. **Ocean** — rescue 5 fish while dodging a shark.
3. **Space** — rescue robots while dodging an alien.

Visuals and characters change with each world, and David's parents show up in matching gear (scuba suits, astronaut suits).

## Details

- Score carries across rounds — faster rescues = more points
- An arrow points you toward the current goal (bear, then home)
- HUD shows time, current round, score, and status
